## 作品简介

《船政遗响》是一页式轻探索网页作品，以马尾船政文化为主题，把“制度、学堂、船坞、航海视野”重构成可点击、可归档、可推进的互动叙事体验。

用户会在五个地点之间切换固定视角，通过“进入场景 -> 靠近器物 -> 收集档案 -> 完成归档”的方式，重新理解船政如何把近代工程教育、制造体系与向海精神连接成一体。

## 在线体验

- 正式地址: [https://oszjh.com](https://oszjh.com)

## 核心亮点

- 五个地点的固定视角探索: 衙门、学堂、船坞、罗星塔、归档室
- 15 份档案收集与归档推进
- 底部对话框式归档反馈，更接近轻游戏叙事体验
- 桌面端与移动端自适应布局
- 纯静态实现，方便部署与展示

## 技术结构

- `index.html`: 页面结构与各类 UI 面板
- `styles.css`: 视觉系统、响应式布局、动画与组件样式
- `script.js`: 场景数据、视角切换、档案收集、归档与结局逻辑
- `output/imagegen/ark/`: 场景图与器物素材
- `tools/ark_image_gen.py`: 素材生成脚本

## 本地预览

直接打开 `index.html` 即可预览。

如果你希望更稳定地本地查看资源加载，也可以在项目目录启一个简单静态服务器后再访问。

## 项目方向

这个作品不是传统信息页，而是更偏“数字展陈 + 轻探索叙事”的表达方式。它尝试回答的核心问题是：

> 船政留下的，究竟只是旧时代的记忆，还是一种至今仍有效的工程方法与向海视野？
